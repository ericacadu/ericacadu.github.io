Callback Function

Callback Function 和一般函式沒有什麼不同，差別在於被呼叫執行的時機。
JavaScript 是一個「事件驅動」的程式語言，也就是說每段程式都必須滿足一定的驅動條件，才會啟動執行程式。

所謂的「Callback Function」其實就是「把函式當作參數，透過另外一個函式呼叫」。
可以解釋成：希望在某個 地點，透過某個 動作，執行某個 程式，來達成結果。
